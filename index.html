Griffin Hagan Project

Working with Day Hagan Asset Management, unique financial research tools were built to produce a robust trading model, aiming to improve on both volatility and returns of the stock market benchmark.

  • C++ for computation
  • Postgres for data organization
  • Golang for data services
  • Scripted in Python 3
  • SVG graphics
  • Jupyter Notebook integration
  • Nginx/Linux for web delivery
  • Angular (1) for web forms
