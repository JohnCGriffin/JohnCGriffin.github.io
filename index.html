Griffin Hagan Research Project

Working with Day Hagan Asset Management (2014-2016), unique financial research tools were built to produce a robust trading model of First Trust sector ETFs, aiming to improve both volatility and returns of the stock market benchmark.

If you have investment interest in this fund, contact Art Day of Day Hagan Asset Management.

The architectural overview describes the how the following technologies were integrated to balance high performance with ease of use by researchers.

  • C++ computation
  • Postgres data organization
  • Golang data services
  • Python 3 Scripting
  • SVG graphics
  • Jupyter Notebook integration
  • Angular (1) web forms
  • Nginx/Linux web delivery
  • LXC containers
